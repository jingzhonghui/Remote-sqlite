## 1. 项目概述

### 1.1 项目背景

RemoteSQLite 是一款跨平台桌面应用，旨在解决远程服务器上 SQLite 数据库的管理问题。传统方式需要将数据库文件下载到本地操作，本项目通过 SSH 协议直接连接远程服务器，调用远端 `sqlite3` 命令行工具执行操作，无需传输数据库文件。

### 1.2 核心目标

- 通过 SSH 安全连接远程服务器
- 在不下载数据库文件的前提下直接操作远端 SQLite
- 提供友好的图形界面完成数据库的增删改查
- 提供 SQL 编辑器，支持语句编写与即时执行
- 提供可视化建表向导

### 1.3 目标用户

| 角色 | 描述 | 主要操作 |
|------|------|----------|
| 普通用户 | 只读查询、数据浏览 | 连接服务器、查询数据 |
| 开发者 | 完整 CRUD、建表、执行 SQL | 所有功能 |
| 管理员 | 管理连接配置、权限 | 连接管理 |

### 2.1 整体架构

```

│  ┌─────────────┬─────────────┬─────────────┬─────────────┐  │

│                          │ IPC 通信                          │
├──────────────────────────┼──────────────────────────────────┤
│                    Electron 主进程                          │
│  ┌───────────────────────┼───────────────────────────────┐  │
│  │        SSH Service    │     SQLite Service             │  │
│  │   (ssh2 客户端封装)    │   (sqlite3 命令包装)           │  │

┌──────────────────────────┼──────────────────────────────────┐
│                    远程服务器                                 │
│                  sqlite3 命令行工具                            │
│                  SQLite 数据库文件                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 分层职责

| 层级 | 职责 | 技术实现 |
|------|------|----------|
| GUI 层 | 用户界面渲染、交互响应 | React + TypeScript + Tailwind CSS |
| 状态层 | 应用状态管理、持久化 | Zustand |
| IPC 层 | 主进程与渲染进程通信桥接 | Electron IPC |
| 服务层 | 业务逻辑、远程命令执行 | ssh2 库封装 |
| 传输层 | SSH 加密通信 | SSH 协议 |

### 2.3 通信流程

1. 用户在 GUI 发起操作（如查询数据）
2. 渲染进程通过 `contextBridge` 调用 `window.electronAPI`
3. IPC 通道将请求转发至主进程
4. 主进程调用 `SSHService` 执行远程命令
5. 通过 SSH 在远程服务器执行 `sqlite3 -json "<sql>"`
6. 捕获标准输出并解析为结构化数据
7. 沿原路返回结果至 GUI 层渲染

## 5. 核心模块设计

### 5.1 SSH 服务模块 (sshService.ts)

**类定义**：`SSHService extends EventEmitter`

**核心属性**：
```typescript
private connections: Map<string, Connection>
```

**Connection 接口**：
```typescript
interface Connection {
  id: string           // 连接唯一标识
  config: SSHConfig    // 连接配置
  client: SSHClient     // ssh2 Client 实例
  status: 'connected' | 'connecting' | 'disconnected' | 'error'
  connectedAt?: Date   // 连接建立时间
  errorMessage?: string
}
```

**SSHConfig 接口**：
```typescript
interface SSHConfig {
  id?: string
  name: string         // 连接名称
  host: string         // 主机地址
  port: number         // 端口，默认 22
  username: string     // 用户名
  authType: 'password' | 'privateKey' | 'agent'
  password?: string
  privateKey?: string
  passphrase?: string
  useAgent?: boolean
  jumpHost?: SSHConfig // 跳板机配置
}
```

**核心方法**：

| 方法名 | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| `testConnection` | `config: SSHConfig` | `Promise<{ success, message }>` | 测试连接是否可用 |
| `connect` | `config: SSHConfig` | `Promise<{ success, connectionId, message }>` | 建立 SSH 连接 |
| `disconnect` | `connectionId: string` | `Promise<{ success }>` | 断开指定连接 |
| `disconnectAll` | - | `Promise<void>` | 断开所有连接 |
| `listConnections` | - | `Array<{ id, name, host, status }>` | 列出所有连接 |
| `execCommand` | `connectionId, command` | `Promise<{ stdout, stderr, code }>` | 在远程执行命令 |

**认证方式处理**：
```typescript
private connectClient(client: SSHClient, config: SSHConfig): void {
  const connectConfig = {
    host: config.host,
    port: config.port,
    username: config.username,
    readyTimeout: 20000,
  }

  // 密码认证
  if (config.authType === 'password' && config.password) {
    connectConfig.password = config.password
  }
  // 私钥认证
  else if (config.authType === 'privateKey' && config.privateKey) {
    connectConfig.privateKey = config.privateKey
    connectConfig.passphrase = config.passphrase
  }
  // SSH Agent 认证
  else if (config.authType === 'agent' || config.useAgent) {
    connectConfig.agent = process.platform === 'win32' ? 'pageant' : process.env.SSH_AUTH_SOCK
  }

  client.connect(connectConfig)
}
```

### 5.2 SQLite 服务模块 (sqliteService.ts)

**类定义**：`SQLiteService`

**依赖**：依赖 `SSHService` 执行远程命令

**核心方法**：

| 方法名 | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| `execute` | `connectionId, sql` | `Promise<{ success, affectedRows?, message? }>` | 执行 DDL/DML 语句 |
| `query` | `connectionId, sql` | `Promise<{ success, columns, rows, message? }>` | 执行查询语句 |
| `getTables` | `connectionId, dbPath` | `Promise<{ success, tables, message? }>` | 获取所有表 |
| `getTableInfo` | `connectionId, dbPath, tableName` | `Promise<{ success, columns, message? }>` | 获取表结构 |
| `getIndexes` | `connectionId, dbPath, tableName` | `Promise<{ success, indexes, message? }>` | 获取索引信息 |
| `getTableDDL` | `connectionId, dbPath, tableName` | `Promise<string>` | 获取建表 DDL |

**数据查询实现**：
```typescript
async query(connectionId: string, sql: string) {
  // 使用 JSON 模式输出，便于解析
  const escapedSql = sql.replace(/'/g, "'\'").replace(/"/g, '""')
  const command = `sqlite3 -json "${escapedSql}"`
  
  const result = await this.sshService.execCommand(connectionId, command)
  
  // 解析 JSON 结果
  let rows: any[] = []
  if (result.stdout.trim()) {
    try {
      rows = JSON.parse(result.stdout)
    } catch {
      // 如果不是 JSON 格式，按行分割
      rows = result.stdout.trim().split('\n').map(line => ({ result: line }))
    }
  }

  const columns = rows.length > 0 ? Object.keys(rows[0]) : []
  return { success: true, columns, rows }
}
```

**获取表结构实现**：
```typescript
async getTableInfo(connectionId: string, dbPath: string, tableName: string) {
  // 使用 PRAGMA 获取表信息
  const sql = `PRAGMA table_info(${this.escapeIdentifier(tableName)})`
  const result = await this.query(connectionId, sql)
  
  if (!result.success) {
    return { success: false, columns: [], message: result.message }
  }

  const columns: ColumnInfo[] = result.rows.map(row => ({
    cid: row.cid,
    name: row.name,
    type: row.type,
    notnull: row.notnull,
    dflt_value: row.dflt_value,
    pk: row.pk,
  }))

  return { success: true, columns }
}
```

## 6. API 接口规范

### 6.1 SSH 相关接口

#### 6.1.1 测试连接
```typescript
ssh.testConnection(config: SSHConfig): Promise<{
  success: boolean
  message: string
}>
```
- **功能**：在不建立持久连接的情况下测试 SSH 连通性
- **超时**：10 秒
- **返回值**：`success=true` 表示连接成功

#### 6.1.2 建立连接
```typescript
ssh.connect(config: SSHConfig): Promise<{
  success: boolean
  connectionId: string
  message: string
}>
```
- **功能**：建立持久 SSH 连接
- **超时**：30 秒
- **返回值**：`connectionId` 用于后续操作标识

#### 6.1.3 断开连接
```typescript
ssh.disconnect(connectionId: string): Promise<{
  success: boolean
}>
```

#### 6.1.4 列出连接
```typescript
ssh.listConnections(): Promise<Array<{
  id: string
  name: string
  host: string
  status: 'connected' | 'connecting' | 'disconnected' | 'error'
}>>
```

### 6.2 SQLite 相关接口

#### 6.2.1 执行 SQL
```typescript
sqlite.execute(connectionId: string, sql: string): Promise<{
  success: boolean
  affectedRows?: number
  message?: string
}>
```
- **适用**：INSERT、UPDATE、DELETE、CREATE、ALTER 等语句
- **注意**：不返回查询结果数据

#### 6.2.2 查询数据
```typescript
sqlite.query(connectionId: string, sql: string): Promise<{
  success: boolean
  columns: string[]
  rows: any[]
  message?: string
}>
```
- **适用**：SELECT 语句
- **返回格式**：JSON 格式，便于前端解析

#### 6.2.3 获取表列表
```typescript
sqlite.getTables(connectionId: string, dbPath: string): Promise<{
  success: boolean
  tables: string[]
  message?: string
}>
```

#### 6.2.4 获取表结构
```typescript
sqlite.getTableInfo(connectionId: string, dbPath: string, tableName: string): Promise<{
  success: boolean
  columns: ColumnInfo[]
  message?: string
}>

interface ColumnInfo {
  cid: number        // 列 ID
  name: string      // 列名
  type: string      // 数据类型
  notnull: number   // 是否非空 (0/1)
  dflt_value: any   // 默认值
  pk: number        // 是否主键 (0/1)
}
```

#### 6.2.5 获取索引
```typescript
sqlite.getIndexes(connectionId: string, dbPath: string, tableName: string): Promise<{
  success: boolean
  indexes: IndexInfo[]
  message?: string
}>

interface IndexInfo {
  name: string
  unique: number
  origin: string
  partial: number
  columns: string[]
}
```

## 8. 状态管理

### 8.1 Zustand Store 设计

**存储划分**：
| 状态 | 持久化 | 说明 |
|------|--------|------|
| savedConnections | ✅ | 用户保存的连接配置 |
| activeConnection | ❌ | 当前活动连接（内存） |
| currentDatabase | ❌ | 当前数据库（内存） |
| selectedTable | ❌ | 选中表（内存） |
| sqlHistory | ❌ | SQL 历史（内存，最多 100 条） |
| savedQueries | ✅ | 保存的查询语句 |
| currentTableDesign | ❌ | 表设计状态（内存） |

**持久化配置**：
```typescript
persist(
  (set) => ({ /* state */ }),
  {
    name: 'remote-sqlite-storage',
    partialize: (state) => ({
      savedConnections: state.savedConnections,
      savedQueries: state.savedQueries,
    }),
  }
)
```

### 8.2 状态更新模式

```typescript
// 模式 1：直接更新
setActiveConnection(connection)

// 模式 2：函数式更新（基于当前状态）
addSqlHistory: (history) =>
  set((state) => ({
    sqlHistory: [history, ...state.sqlHistory.slice(0, 99)], // 最多保留 100 条
  }))

// 模式 3：条件更新（防重）
saveQuery: (name, sql) =>
  set((state) => ({
    savedQueries: [
      ...state.savedQueries.filter((q) => q.name !== name),
      { name, sql }
    ],
  }))
```
